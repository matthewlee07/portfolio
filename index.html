Intro. to Programming

There was a matchmaker, Facebook Ads, who first introduced me to programming. It was an advertisement for one of the many programming bootcamps.


Even you can become a full stack developer within 3 months!
No programming experience necessary!
Get a $100k+ job immediately!
No tuition payments until you've landed a job!

So I began my courtship. The first time I saw code was around January 2017. I bought books, did tutorials through Codecademy, and by March I was able to solve some Code Wars problems.
I attended an in-person pre-bootcamp in San Francisco for two weeks, returned to Columbus for additional practice, and after considering the many options, enrolled in Thinkful's online bootcamp by August.

These past few months have been bumpy, but I'm glad to say my relationship with programming progresses and I couldn't be happier. Programming always seemed like magic to me. The mystery has been unveiled, yet it still amazes me that so much can be built with some logic, coffee, and a lot of Stack Overflow.
